### üöÄ GETTING STARTED

1.  **Run the script for the first time.**
    - `python MUZ_REFORGED.py`
    - It will ask for your **main account token**. Paste it in and press Enter.
    - It will ask for a **prefix**. Use something like `.` or `!`. Press Enter.
    - This creates `config.json`. You only need to do this once.

2.  **Add Multi-Client Tokens (Optional)**
    - A file named `tokens.txt` will be created.
    - Open it and paste in your alt tokens, **one token per line**.
    - These are used for commands like `.kill`. If you don't have any, those commands won't work.

3.  **Run the Bot**
    - Run `python MUZ_REFORGED.py` again.
    - It will now log in using the token from `config.json`.
    - The terminal will show a fancy startup screen if it works.
